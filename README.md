# Zen Profile

A beautiful native profile manager for Zen Browser.

## Features

- 🚀 Launch any Zen Browser profile
- 👤 Custom avatars
- 💾 Persistent profile settings
- ⚡ Native performance with Rust + Tauri
- 🎨 Modern UI built with React
